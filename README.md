# 読書メモ
読書習慣をつけたい人、読書を振り返りたい人のための  シンプルな『読書記録アプリ』です。<br>
探す→保存→記録の3ステップで、自分だけの読書記録を残せます。<br>
(レスポンシブ対応可)<br>
![トップ](src/assets/img/reading-memo.png)

# 作成した目的
読書をただ読んで終わらすのではなく、簡単に自分のメモや感想を残せるものを作りたいと思いました。<br>
ユーザーが継続しやすいよう、日々の読書量を可視化できる作りになっています。
